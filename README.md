# Lottery 6/36 – WinForms Application

## Description
This is a simple Windows Forms application that simulates a 6/36 lottery draw.
The user can draw 5 main lottery balls and 1 additional ball.
Each number is drawn randomly and cannot be drawn more than once.

## Features
- Draws random lottery numbers in the range 1–36
- Prevents duplicate numbers
- Draws 5 main balls and 1 extra ball
- Displays the final result as a sorted 5 + 1 combination
- Reset functionality to start a new draw
- Clear and simple user interface

## How to Run
1. Open the solution in Visual Studio
2. Build the project
3. Run the application

## Technologies Used
- C#
- .NET 9.0
- Windows Forms
